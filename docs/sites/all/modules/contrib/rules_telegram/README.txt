Rules Telegram
https://www.drupal.org/project/rules_telegram
=============================================

INTRODUCTION
------------

Send notifications to telegram via Rules.

REQUIREMENTS
------------

rules

INSTALLATION
------------

* Install as you would normally install a contributed Drupal module. Visit:
  https://drupal.org/documentation/install/modules-themes/modules-7
  for further information.

CONFIGURATION
-------------

Go to admin/config/workflow/rules_telegram and set proxy server for telegram
(if needed)

Go to admin/config/workflow/rules and set your rules
