INTRODUCTION
------------

This module uses free weather data from yr.no to display current weather
conditions from anywhere in the world. Data for more than 14.000 places
worldwide is included for easy weather display.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/weather

 * To submit bug reports and feature suggestions, or track changes:
   https://www.drupal.org/project/issues/weather


REQUIREMENTS
------------

No special requirements.


RECOMMENDED MODULES
-------------------

 * Location (https://www.drupal.org/project/location):
   When enabled, display the weather nearby a node's location.
 * Smart IP (https://www.drupal.org/project/smart_ip):
   When enabled, display the weather near the user's location.
 * Geofield (https://www.drupal.org/project/geofield):
   When enabled, display the weather near a node's location or
   near a taxonomy term's location.


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit
   https://www.drupal.org/documentation/install/modules-themes/modules-7
   for further information.
