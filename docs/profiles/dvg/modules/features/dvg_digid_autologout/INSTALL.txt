INSTALL.txt file
================
$Id: INSTALL.txt,v 1.2 2009/04/17 14:05:51 jvandervort Exp $

1. Place the dvg_digid_autologout folder in your drupal/modules folder (or if multisite and this
  module is meant to be available to just one site in drupal/sites/www.YOURSITE.com/modules)

2. Enable the dvg_digid_autologout module at administer > modules

3. Set-up the module at administer > configure > services > DigiD - Auto Logout
