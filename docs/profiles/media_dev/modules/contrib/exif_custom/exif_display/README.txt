EXIF Data Display
-----------------
This add-on for the EXIF Custom module provides a pseudo / "extra" field on the
image file entity to display all available non-empty values.


Features
--------------------------------------------------------------------------------
* Provides an "extra" field tied to the image file entity.
* Displays all available EXIF meta data in a table display.
* Compatible with the Field Group module to improve the editorial UX:
  https://www.drupal.org/project/field_group


Requirements
--------------------------------------------------------------------------------
* EXIF Custom
* File Entity
